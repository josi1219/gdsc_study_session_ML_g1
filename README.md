# GDSC ML Study Session - Group 1

## Overview
This repository contains the submission for **Week 1 of the GDSC Machine Learning Study Session (Group 1)**. The task focuses on basic **Python** and **NumPy** operations by performing calculations on a list of numbers.

---

## Task
The assignment includes:
1. Creating a list of 5 numbers in Python.
2. Converting the list into a NumPy array.
3. Calculating and displaying:
   - Mean
   - Maximum value
   - Sum
4. Printing the results clearly.

---

## Technologies Used
- Python 3
- NumPy

---

## How to Run
1. Clone the repository:
```
git clone https://github.com/josi1219/gdsc_study_session_ML_g1.git
```
2. Navigate to the folder:
```
cd gdsc_study_session_ML_g1
```
3. Install NumPy if not already installed:
```
pip install numpy
```
4. Run the script:
```
python numpy_task.py
```
5. The console will display the **mean, maximum, and sum** of the numbers.
